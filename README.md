# A simple Vi-like TUI text editor on Rust.
Dependencies: ratatui, crossterm, mlua latest
## Current features:
- Command/insert mode
- Simple command parses
- Navigating in buffer using keyboard arrows, mouse wheel; pgup/pgdn/hm/end.
- Tabs
- Command aliases
